#Setting up your environnement

The Software in this directory is arduino compatible.
It is intended to be used withe arduino ide that you can download there:
https://www.arduino.cc/en/Main/Software
***
This software is delivered for the V1.0 version of the yred minikame board.
The board is compatible with the NodeMCU V1.0 so it is recommended that your
IDE setting match this board.
it is recomended that you follow this lin in order to set up the environnement:
http://www.instructables.com/id/Quick-Start-to-Nodemcu-ESP8266-on-Arduino-IDE/
***
The Yred minikame board is using a FT230X UART to USB chip from FTDI.
If your system doesn't install automatically the driver when you plug your robot
please follow this link in order to manually install the driver :
http://www.ftdichip.com/Products/ICs/FT230X.html
